# PokAPI

Une application web permettant aux utilisateurs de **créer un compte**, **capturer des Pokémon**, et **construire des équipes** en utilisant l'API [PokeAPI](https://pokeapi.co/).

---

## 🎯 Fonctionnalités

- **Authentification** : Inscription et connexion sécurisée.
- **Capture de Pokémon** : Ajoute des Pokémon à ta collection personnelle.
- **Création d'équipes** : Constitue des équipes de 6 Pokémon.
- **Exploration** : Parcoure la liste des Pokémon avec leurs détails (types, statistiques, évolutions).
- **Gestion de collection** : Organise et consulte tes Pokémon capturés.

---

## 🛠 Technologies utilisées

Frontend : Développé avec Vue.js 3 pour une interface réactive et moderne.
Backend : Construit avec Node.js et Express pour gérer les requêtes API.
Base de données : MongoDB pour stocker les données utilisateur et les informations complémentaires.
